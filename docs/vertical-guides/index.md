# Vertical Guides

These guides translate the core framework into role-specific workflow patterns.
Each guide links low-value task clusters to the canonical scorecard and a
matching design-spec example.

## Seed verticals

- professional services
- SaaS
- ecommerce

## How to use the guides

1. Start with the role that owns the workflow bottleneck.
2. Use the guide to identify task clusters worth inventorying.
3. Score the tasks using the [canonical rubric](../task-scoring/canonical-task-rubric.md).
4. Open the linked example spec to see one possible design boundary.
